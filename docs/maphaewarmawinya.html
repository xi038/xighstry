"ဟာ...ဆရာ ဆရာ ”

ပြေးလာကြတယ်။

"ဝေါ့ ဝေါ့ ”

ရောဗျို့။

"ဟာ အောင်မလေး လုပ်ကြပါဦးဗျ”

"ပြေးကြဟေ့ ၊ ပြေးကြ ဒါမယ်ရီ မဟုတ်ဘူး။ မဖဲဝါကြီးတဲ့ဟဲ့”

"ဘုန်း ဘုန်း ဘုန်း ဘုန်း ဘုန်း ”

ဟာ သွားပြီဗျာ။ ထွက်ပြေးတဲ့ လူတွေကို မဖဲဝါလိုက်ရိုက်တာ

အရိုက်ခံရတဲ့ လူတိုင်း မြေကြီးပေါ်မှာ ဆန့်ငင် ဆန့်ငင်

လူတွေ နောက်ကို မဖဲဝါ ပြေးလိုက်ပြီး ရိုက်တယ်။

"ဘုန်း ဘုန်း ဘုန်း”

တစ်ရွာလုံး အော်သံဟစ်သံ ပြေးသံ ငိုသံ ညည်းညူသံ

ပါဘူးဗျာ။

ကျုပ်တစ်သက်မှာ ဒီလောက်များတဲ့ လူသေအလောင်းတွေကို

ဟာ ဟိုမှာ မယ်ရီဗျို့၊ မယ်ရီ လူသေတွေကို ကျော်ခွပြီး

သွားနေတာ ။ဆံပင်ဖားလျားကြီး ချလို့ဗျ။ တကယ်တော့

"ဟေ့ ရပ်လိုက်စမ်း”

အသံကြီးတစ်သံဗျ။ ဒါပေမဲ့ ကြည်လင်ပြီး သြဇာဓာတ်

လိုက်တယ်။ မြင်းဖြူကြီးတစ်ကောင်ကို စီးပြီး ရွာထဲကို

ဝင်လာတဲ့ လူဗျာ။မြင်းပေါ်က လူကို ကျုပ်သေသေချာချာ.

"မဖဲဝါ ကြားသိစေရမည် ။ အထက်ဆရာကြီးတွေရဲ့ အမိန့်၊

ငါဆရာ အဖေရှင်ရဲ့ အမိန့်”

ဟော....မဖဲဝါ ချက်ချင်းဒူးထောက်ချလိုက်တယ်။

"နောက်နောင်မှာ ဒီလိုအလုပ်မျိုး လုံးဝမလုပ်ရ ၊ လူတွေနေတဲ့

မြို့ရွာအတွင်းကို လုံးဝမဝင်ရ ။ သင်္ချိ ုင်းထဲမှာသာ သင်နေထိုင်

ရမည်။ သင်္ချိ ုင်းမှန်သမျှ သင်အပိုင်စားစရာ ဖြစ်ရမည်။

အထက်ပုဂ္ဂိုလ်တို့ရဲ့အမိန့်ကို ဖီဆန်ခဲ့ပါက

"ဘုန်း”

"လူလေးနိုးပြီလား ၊ ခေါင်းထဲမှာ ကြည်ကြည်လင်လင်ရှိရဲ့

လား”
